# Lotto Number Recommender

## Overview

This project is a simple web application that generates and displays six random lotto numbers between 1 and 45. It now includes a dark mode feature for better accessibility and user preference.

## Features

*   **Number Generation:** Generates six unique random numbers from 1 to 45.
*   **Display:** Displays the generated numbers in a clear and organized way.
*   **User Interface:** Provides a simple and intuitive user interface with a button to generate numbers.
*   **Dark Mode:** A toggleable dark mode that persists across sessions.

## Project Structure

*   `index.html`: The main HTML file, now includes a dark mode toggle button.
*   `style.css`: The CSS file, refactored to use CSS variables for light and dark modes.
*   `main.js`: The JavaScript file, now handles dark mode logic and localStorage persistence.
