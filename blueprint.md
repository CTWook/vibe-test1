# Lotto Number Recommender

## Overview

This is a web application that recommends lotto numbers.

## Implemented Features

*   Initial project setup.
*   **UI Implementation**: Created a user interface to display the recommended lotto numbers.
*   **Number Generation Logic**: Implemented the logic to generate 6 unique random numbers between 1 and 45.
*   **Display Logic**: Implemented the logic to display the generated numbers on the screen.
